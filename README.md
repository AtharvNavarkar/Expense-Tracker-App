Expense Tracker App

This is a simple Expense Tracker web application made using JavaScript.
It helps users add their daily expenses, view the list, delete expenses, and see the total amount spent.
All data is saved in the browser using LocalStorage.

Features
  Add new expenses with name and amount
  Display list of all expenses
  Delete any expense from the list
  Automatically calculate total amount
  Data is saved even after page refresh
  Simple and clean user interface

Technologies Used  
  HTML – Page structure
  CSS – Styling
  JavaScript – Logic and DOM manipulation
  LocalStorage – To store expense data

How the App Works
  User enters expense name and expense amount.
  User clicks Add (submit button).
  Expense is added to the list and saved in LocalStorage.
  Total amount is updated automatically.

User can click Delete to remove any expense.

When page is refreshed, all expenses are still available.
